# Business Automation Prompts

## Lead Follow-Up
```
Check Airtable for leads with status "Waitlist" that signed up more than 48 hours ago
but haven't received a follow-up. Draft a personalized email for each using their
name and package interest. Don't send yet — show me the drafts first.
```

## Invoice Reminder
```
Check for any unpaid invoices older than 14 days. Draft a friendly but firm reminder
email for each client. Include the invoice amount and original due date.
Tone: professional, not aggressive.
```

## Weekly Report
```
Generate my weekly business report:
1. New leads this week (from Airtable)
2. Emails sent/received count
3. Upcoming meetings next week
4. Any overdue tasks
Format as a clean summary I can skim in 2 minutes.
```

## Competitor Check
```
Search the web for recent news, blog posts, and social media activity from these competitors:
[LIST YOUR COMPETITORS]
Summarize anything interesting: new features, pricing changes, content themes, hiring signals.
```

## Client Onboarding
```
New client onboarding for {{CLIENT_NAME}}:
1. Create Airtable record with their details
2. Set up project folder structure
3. Draft welcome email with next steps
4. Add kickoff meeting to calendar for next week
5. Create project brief template
```

## End of Day Summary
```
Summarize everything that happened today:
- Emails received and sent
- Leads captured
- Tasks completed
- Any issues or errors
- What needs attention tomorrow
Save to memory/YYYY-MM-DD.md
```
