# Customer Support Prompts

## Inquiry Response
```
A potential client just messaged asking about {{SERVICE}}.
Draft a response that:
- Acknowledges their specific need
- Briefly explains how we can help
- Shares one relevant example/case study
- Suggests a next step (call, proposal, demo)
- Matches our brand voice: {{YOUR_VOICE}}
Don't send — show me first.
```

## FAQ Handler
```
Someone asked: "{{QUESTION}}"
Check our docs, website, and previous conversations for the answer.
If we've answered this before, use that answer (updated if needed).
If it's new, draft a helpful response and suggest adding it to our FAQ.
```

## Complaint Resolution
```
Client {{NAME}} is unhappy about {{ISSUE}}.
1. Acknowledge the problem empathetically
2. Explain what happened (check our records)
3. Propose a concrete solution
4. Draft the response
Tone: humble, solution-oriented, no excuses. We own our mistakes.
```

## Proposal Follow-Up
```
Check Offorte for proposals sent more than 5 days ago that haven't been viewed or signed.
For each:
- Draft a casual follow-up message
- Reference something specific from their project
- Don't be pushy — just checking in
- Suggest a specific time to chat if they have questions
```

## Onboarding Welcome
```
New client {{NAME}} just signed up for {{PACKAGE}}.
Send them:
1. Welcome email with next steps
2. Calendar link for kickoff call
3. Intake form link (if applicable)
4. Set their Airtable status to "Onboarding"
5. Add a follow-up reminder for 48 hours
```
