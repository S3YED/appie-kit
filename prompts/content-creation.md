# Content Creation Prompts

## Blog Post Draft
```
Write a blog post about {{TOPIC}} for my audience of {{TARGET_AUDIENCE}}.
Tone: {{YOUR_BRAND_VOICE}}
Length: 800-1200 words
Include: Hook, 3-5 key points, real examples, CTA
SEO keywords: {{KEYWORDS}}
Don't publish — save as draft for my review.
```

## Social Media Batch
```
Create a week of social media content (7 posts) for {{PLATFORM}}:
- Topic theme: {{THEME}}
- Mix: 2 educational, 2 behind-the-scenes, 2 engagement, 1 promotional
- Include hashtag suggestions
- Format for {{PLATFORM}} (character limits, image suggestions)
- Tone: {{YOUR_VOICE}}
```

## Video Script (Reel/Short)
```
Write a 30-second script for an Instagram Reel about {{TOPIC}}.
Structure:
- Hook (first 3 seconds, stop the scroll)
- Problem (5 seconds)
- Solution (10 seconds)
- Proof/example (7 seconds)
- CTA (5 seconds)
Tone: energetic, direct, no fluff.
```

## Email Newsletter
```
Draft this week's newsletter:
- Subject line (A/B test: 2 options)
- Preview text
- Opening hook (personal, not corporate)
- Main content: {{THIS_WEEK_TOPIC}}
- One valuable tip or resource
- CTA: {{DESIRED_ACTION}}
Keep it under 500 words. Nobody reads long emails.
```

## Case Study
```
Write a case study about our work with {{CLIENT}}:
1. The problem they had
2. What we built/did
3. The results (real numbers)
4. Client quote (draft one they can approve)
5. Key takeaways
Format: Problem → Solution → Results
```
